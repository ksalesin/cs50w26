# Welcome!

In this course you will learn how to design and build large software systems. You'll learn how to use the Linux operating system and software development tools, how to program in the shell and in C, and learn how to craft software that is understandable, efficient, testable, reliable, and secure.

Please see the [syllabus](/syllabus) for class details.
